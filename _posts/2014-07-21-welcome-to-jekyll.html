提交代码：

  1. git add .
  2. git commit -m "YOUR COMMENT"
  3. git push origin gh-pages

删除文件

  1. git rm -rf 目录名
  2. git commit -m "YOUR COMMENT"

revert文件

  1. git log index.js
  2. 在log列表查到commit的id，例如 c6c8f555535e2360a1b5576c9f5d907f7d45723c
  3. git reset c6c8f555535e2360a1b5576c9f5d907f7d45723c index.js
  4. 编辑

错手add了怎么办？

  1. git reset即可

这里有一份非常棒的入门教程，很适初学者快速入门《git - 简明指南》

如何在github上仅下载一部分文件（原理未知，亲测可用）

  1. 打开SVN（没看错，就是SVN）
  2. 复制需要的项目目录路径
  3. 将目录路径中'/tree/master/'改成'/trunk/'
  4. checkout即可

windows本地jekyll安装

  1. 下载ruby
  2. gem install jekyll(这时候发生错误：Could not find a valid gem 'jekyll' (>= 0), here is why...)，解决方法是下载认证文件，并放在ruby的bin目录下，貌似还需要等一会儿(30分钟内)才能够生效
  3. 安装完以后，按照官网教程命令
    jekyll new my-site
    cd my-site
    jekyll serve --watch(加上watch参数可以实时看到代码更新)
    但是发现命令行报错，在stackoverflow上查到了解决办法，就是使用 ruby v 1.9.3 并且安装rouge，即可~终于可以本地调试jekyll了~
